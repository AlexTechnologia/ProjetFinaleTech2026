# VeilCraft — Asset drop-in folder (CC0)

The engine already runs with improved procedural models. To upgrade to real
low-poly art, download the CC0 packs below and drop the files here. No code
changes are required once the glTF loader hookup (next phase) is in place.

## 3D models  -> assets/models/  (.glb)
- Vegetation & rocks: Quaternius Ultimate Nature Pack
  https://quaternius.com/packs/ultimatenature.html  (CC0)
- Crafting stations & ore: Kenney Survival Kit
  https://kenney.nl/assets/survival-kit  (CC0)
- Player characters (rigged + animated): Quaternius Ultimate Animated Character Pack
  https://quaternius.com/packs/ultimatedanimatedcharacter.html  (CC0)
- Enemies: Kenney Animated Characters (Survivors)
  https://kenney.nl/assets/animated-characters-survivors  (CC0)

Suggested filenames the loader will look for:
  models/tree_oak.glb, tree_birch.glb, tree_dark_oak.glb,
  models/rock.glb, iron_ore.glb, flint.glb,
  models/player.glb, enemy_goblin.glb, enemy_orc.glb, enemy_knight.glb

## 2D inventory icons  -> assets/icons/  (.png, 64x64)
- Kenney Game Icons: https://kenney.nl/assets/game-icons  (CC0)
- Ultimate Low Poly Survival Icons:
  https://opengameart.org/content/ultimate-low-poly-survival-icons  (CC0)

Name icons by item id, e.g. icons/wood.png, icons/iron_ore.png, icons/cooked_meat.png
